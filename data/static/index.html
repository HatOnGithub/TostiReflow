About

TostiRelfow is a project to control an oven using a microcontroller.
It allows you to create custom profiles for reflow soldering.
It is designed to be simple and easy to use, with a focus on flexibility and customisation.
This project was started after I found out that there were no reflow ovens available at LPL.
Due to the high cost of commercial reflow ovens, I decided to create my own for future makers interested in SMT.
